字符串类型

字符串是 Python 数据类型的一种，就是由零个或多个字符组成的有限序列，从名字我们就可以知道字符串就是一串
字符，在 Python 中被一对引号括起来的就是字符串，引号可以是单引号，也可以是双引号，但是必须是英文下的引号。

注释

通常使用井符号 # 将代码屏蔽掉，意思就是井号后面的内容都会被Python解释器忽略。这样的功能
叫 注释 。不同的编程语言，注释的标识可能不同。

转义字符 \

“转义字符” 顾名思义就是能将字符意思改变的符号，转义字符 \ 可以转义很多字符，比如 \n 表示换行， \t 表示
制表符，字符 \ 本身也要转义，所以 \ 表示的字符就是 \ 。

如果字符串里面有很多字符都需要转义，就需要加很多 \ ，为了简化，Python还允许用 r”” 表示 “” 内部的字符串
默认不转义

三重引号

如果字符串内部有很多换行，用 \n 写在一行里不好阅读，为了简化，Python允许用 ‘’’…’’’ 或 “””…””” 的格
式表示多行内容

在使用print()进行输出的时候，输出的结果总是自动换行，因为在print()函数中，有一个默认的参数 end=\n ，如果不
希望输出的结果直接换行，可以将这个参数改变

获取字符长度

len() 函数获取字符串的长度，len()函数我们会经常使用。

strip()删除开头或结尾空白字符

如果字符串的开头和结尾都有空白字符，那么都会删除

count()获取某个字符在字符串出现的次数
