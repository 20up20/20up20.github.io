元组

Posted on 2020-04-22

元组

Python元组与列表类似，不同之处在于元组的元素无法修改，元组使用小括号 () ，元组是不可变数据类型，不能修
改或删除元素，也不能添加元素。列表使用方括号 [ ] ，另外列表是一个可变类型的数据。

元组也可以容纳不同的数据类型。

元组的创建

使用 () 就可以创建元组，元素之间使用英文逗号隔开

索引

元组也可以使用索引访问元素

切片

元组也可以通过切片获取元素

修改元组中的列表

元组是不可变的类型，前面说过元组中的元素不可修改，但是如果元组中嵌套了列表，那么就可以修改这个列表

注意：这种修改元组的方式是不推荐的，尽量不要使用这种方式，我们默认元组就是不可修改的

元组拆包

1.想要取出元组中的元素，可以使用元组拆包

2.如果元组中存在不想要的元素，可使用占位符

3.如果元组中元素很多可以使用 * 表示任意多个元素（在元组长度范围之内）

其实 a, b = b, a ， a, b, c = 10, “python”, 3 这样的写法就是元组拆包的运用

特别注意

如果一个元组对象中只有一个元素，那么这个元素后面一定要接一个 , （逗号）不然这个对像就不是元组类型

列表

说明

列表是Python中最常用的数据类型
列表中可以容纳不同的数据类型
列表是一种可变序列
列表可以通过索引访问元素（和字符串一样索引是从0开始）
列表支持切片运算

列表创建

使用 [ ] 创建列表，元素之间使用逗号隔开
list()创建列表

乘法运算

乘号表示列表元素的重复

加法运算

使用加法运算可以将列表拼接在一起

len()

使用len()函数可以查看列表的长度

索引

列表是一个可变对象，可以通过索引来改变列表元素的值，也可以通过列表来访问元素

切片

[start:end:step]

1.start 起始位置
2.end 结束位置
3.step 步长, 默认是 1, step 为正数时，从左到右；负数时，从右到左

append()

使用append()方法可以向列表末尾添加元素

pop()

使用pop()方法可以删除列表指定位置的元素，并返回指定位置元素的值，不指定位置则删除末尾元素

extend()

extend()方法一次性向列表末尾添加多个元素，对象可以直接为列表

insert()

insert()方法可以将元素插入到列表的指定位置

in/not in

使用in或者not in可以判断一个元素在不在列表中，返回值为True/False

remove()

remove()用于按值删除元素，找到第一个符合的元素然后删除

del

del 用于删除指定列表中的元素

max()

max()函数用于返回列表最大值

min()

min()函数用于返回列表中的最小值

sum()

sum()函数用于对列表中的元素进行求和，但是列表中的元素必须都是数值类型的

嵌套列表

列表中也可以嵌套列表

在Python中一般不用列表来表示二维数组或者多维数组，虽然Python也能表示，但是性能较低，一般使用array或者numpy来创建多位数组

浮点数

浮点数的变量就是浮点型 (float) 也就是小数，之所以称为浮点数，是因为按照科学记数法表示时，一个浮点数的
小数点位置是可变的。浮点数可以用数学写法。

整数和浮点数在计算机内部存储的方式是不同的，整数运算永远是精确的（除法也是精确），而浮点数运算则可能会有四舍五入的误差。

浮点数也可以通过变量赋值来计算。

Posted on 2020-04-21

布尔类型

布尔类型是计算机中最基本的类型，它是计算机二进制世界的体现，一切都是 0 和 1 。

Python中的布尔类型只有两种值： True 和 False 。

（注意：首字母都是大写，与C++、JavaScript中的小写有所不同）

Python里面实现了一个 类型对 象 叫做 bool ，bool是一个 int 的子类，内置的 True 和 False 就是bool仅有的两个实例对象。

使用 bool() 函数我们就可以对对象进行布尔真假判断，将给定参数转换为布尔类型，参数不为0和空则返回 True，如果没有参数，返回 False。

布尔运算符：or，and，not（优先级为升序）

or 是一种“短路运算符”，只有当第一个为False时才去验证第二个。即：两个变量只要有一个为True则为True。

and 也是种“短路运算符”，只有当第一个为True时才去验证第二个。即：两个变量都为True时结果才为True。

not 的优先级比非布尔运算符底，所以 not a == b 解释为 not (a == b) ，并且 a == not b 是语法错误。

比较运算

布尔值反应的是“是非”，有比较才有是非。

Python中有8种比较运算。它们有相同的优先级，比布尔运算的优先级高。

比较运算符可以任意的连写，比如： x < y <=z 相当于 x < y and y <= z 。

8种不同的比较运算：

1.<:x小于
2.<=:小于等于
3.>:大于
4.>=:大于等于
5.==：等于
6.!=:不等于
7.is:是对象
8.is not:不是对象
